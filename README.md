# EchoCanvas

**AI Thought Visualization Canvas** - See how AI thinks.

EchoCanvas transforms AI reasoning processes into interactive visual maps.
Every analysis, hypothesis, verification, decision, and action becomes a
node on a canvas that you can explore, replay, and understand.

## Features

- **5 Thought Types** - Color-coded nodes: Analysis (blue), Hypothesis (purple),
  Verification (green), Decision (amber), Action (red)
- **Timeline Replay** - Watch thoughts unfold step by step
- **Decision Explorer** - Click any decision node to see alternatives considered
- **Cognitive Map** - Auto-extracted knowledge structure with certainty levels
- **Import Conversations** - Paste any AI conversation to visualize its thinking
- **Dark Theme** - Designed for focused exploration

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

React 19 + TypeScript + Vite + React Flow + dagre + Tailwind CSS + Zustand

## Data Safety

- Pure frontend, no external API calls
- All data stays in your browser (IndexedDB)
- Example data is fully anonymized
- Zero tracking, zero telemetry
